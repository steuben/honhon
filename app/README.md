# 🇫🇷 HonHon - Französisch Lernen

Eine persönliche Französisch-Lern-App für deutsche Muttersprachler, basiert auf wissenschaftlichen Lernmethoden und dem SM-2 Spaced Repetition Algorithmus.

## Features

- ✅ **Spaced Repetition (SRS)** - Intelligentes Wiederholungssystem basierend auf SM-2
- ✅ **Offline-First** - Alle Daten lokal gespeichert mit IndexedDB
- ✅ **Deutsche Oberfläche** - Vollständig auf Deutsch
- ✅ **Aussprache** - Web Speech API + Google Translate Fallback
- ✅ **Fortschritts-Tracking** - Streak-Counter, Genauigkeit, Statistiken
- ✅ **20 Anfänger-Vokabeln** - Pronomen, Artikel, Verben, Nomen
- ✅ **Savegame-System** - Fortschritt wird automatisch gespeichert

## Installation & Start

\`\`\`bash
npm install
npm run dev
\`\`\`

Öffne http://localhost:5173 im Browser

## Tech Stack

- SvelteKit 2.x + TypeScript
- Tailwind CSS
- Dexie.js (IndexedDB)
- Web Speech API

## Verwendung

1. **Dashboard**: Übersicht über Fortschritt
2. **Lernen**: Neue Vokabeln lernen (5 pro Session)
3. **Wiederholen**: Fällige Karten wiederholen
4. **Vokabular**: Alle Wörter durchsuchen

Viel Erfolg beim Französisch lernen! 🇫🇷
